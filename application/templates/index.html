Planning Data Analysis tools

This application is a work in progress. It is a collection of tools for analysing planning data.

It is built using Digital Land planning data analysis tools

Tools

  • Extract tables from a PDF file or a webpage
  • Analyze clusters of invalid planning application reasons
  • Collect planning document data from local authority websites
